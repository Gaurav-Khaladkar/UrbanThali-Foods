# Delivery Roadmap

## Phase 1 - Foundation (Weeks 1-2)

- Repository bootstrap and standards
- Core domain contracts
- Health and readiness APIs
- Environment and deployment baseline

## Phase 2 - Subscription Core (Weeks 3-6)

- Plan catalog
- Customer onboarding
- Pause/skip lifecycle
- Renewal scheduler

## Phase 3 - Kitchen and Dispatch (Weeks 7-10)

- Menu and meal slots
- Kitchen prep manifests
- Rider assignment and route states
- Delivery confirmation workflow

## Phase 4 - Business Control (Weeks 11-14)

- Finance and reconciliation dashboards
- Promo and referral controls
- Operational KPIs and alerts

## Phase 5 - Hardening (Weeks 15-16)

- Security testing
- Load and resilience testing
- Store release readiness
